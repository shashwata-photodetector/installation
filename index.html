Photodetector

Introducing our laboratory-developed photodetector, designed for precision optical measurements. This device is capable of providing both position and light sensitivity information with high accuracy. It operates on a resistive measurement principle, ensuring cost-effective development.

After installing Python, verify the installation by running the following commands in your terminal:

python --version

This command should display the installed Python version.

pip --version

This command should display the installed pip version.

Next, navigate to the command prompt and install all required dependencies using the command pip install [dependencies].
